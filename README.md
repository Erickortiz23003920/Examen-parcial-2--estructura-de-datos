Descripción del Proyecto
Este proyecto consiste en la implementación de una lista doblemente enlazada circular desarrollada en lenguaje Java. Esta estructura de datos lineal se caracteriza porque cada nodo posee dos punteros: uno hacia el nodo siguiente y el otro hacia el nodo anterior, permitiendo un recorrido bidireccional.

Estructura del Proyecto
El proyecto está organizado de la siguiente manera:

Nodo.java: Define la unidad básica de estructura (el nodo). Contiene el tipo de dato enteroy los punteros de autoreferencia.
ListaDobleCircular.java: Contiene la lógica de negocio y los algoritmos para las operaciones de inserción, eliminación, busqueda y recorrido de la lista circular.
Main.java: Implementa el menú interactivo para que el usuario pueda probar todas las funcionalidades en tiempo real.
README.md: Este archivo con la documentación y datos del estudiante.
